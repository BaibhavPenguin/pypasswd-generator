# **Pypasswd Generator** 
The highly secure and reliable alternative for password generation.  
**Pypasswd generator** is a python utility which is made for mass generation of unique and highly secure password combinations using a **Multi Step Valitadion Engine** and real world breach detection.

**Pypasswd generator** is still in development as of June 2026 and docker images as well as zipped Python Scripts will be released.

## **What does Pypasswd Generator do?**  
**Pypasswd generator** automatically generates a dataset highly secure and unbreached password canditates using pesudo-random string generation and a multi step validation engine. It automatically detects common patterns , breached passwords , common dictionary words and basic password formation to output a password which is highly resistant to automated attacks like Brute-Forcing and Logical building.

**Pypasswd generator** uses a simple punishment system to evaluate the score of each test and get a highly accurate evaluation of the password combination being tested. It actively punishes canditates like `its_my_passWOrd_&63_12` 0r `%%2@#PAAS_WOrD_9878_0973_3$^5` , these passwords may look like strong canditates and they are for most uses cases but the passwords generated by **Pypasswd Generator** are even better and surpass these canditates easily ensuring security and maximum resistance from brute force attacks.

For Example : A canditate which is accepted at this stage of development by **Pypasswd Generator** is `#p9&46;7qW1!3nN/0{9=1Pb@Wq;7*gC`  
Now obviously this password is not made to be remembered by a human, but that is intentional as *Pypasswd Generator* is made to function as a backend for password manager and password recommendation utilities in a small to mdeium scale.

# **Pypasswd Validation Engine**  
*The Validation Engine* is a major component of **Pypasswd Generator** and it is crucial for ensuring that generated passwords are secure and brute force resistant.   
**The Overview of the validation process**  

- CSPRNG Input Formation
- Preliminary Tests
- Pattern Matching 
- Breach Detection
- Dictionary Detection
- Output Generation & Cleaning

**CSPRNG Input Formation** : Generates a Cryptographically Secure Random Number. 

**Preliminary Tests** : Classify on Length & Character Set (Digits,Upper case < Lower case,Symbols).  

**Pattern Matching** : Filter combinations on strict parsing rules and detect common patterns.  

**Breach Detection** : Look for exact matches and similar matches in public breached password databases. To ensure that generated output isn't just availabe on some database.  

**Dictionary Detection** : Detect for legible language in English, words like `password` ,`Tiger` , `personal` etc to prevent accurate gueeses and over the shoulder snooping.
